# Food Delivery Product & Courier Analytics

SQL/Python project analyzing product metrics, courier performance, user retention, and delivery operations for a food delivery service.

## Project report

[Open PDF report](report/food_delivery_report.pdf)

## Tools

- SQL
- SQLite
- DBeaver
- Python
- Pandas
- Matplotlib
- SciPy
- Statsmodels

## Key results

- Delivery rate: 95%
- Average delivery time: 19.96 minutes
- 93.01% of orders delivered within 16–30 minutes
- Repeat order rate: 68.25%
- D7 retention: 5.81%
- No statistically significant relationship between courier sex/age group and delivery time
